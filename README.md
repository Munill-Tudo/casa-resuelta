# Casa Resuelta

Web editorial de afiliados para hogar práctico: productos útiles, decisiones sencillas.

## Comandos

```bash
npm install
npm run build
npm run dev
```

## Estado V1

- 30 contenidos iniciales generados desde `src/data/articles.ts`.
- Categorías desde `src/data/categories.ts`.
- Enlaces de Amazon en modo placeholder hasta configurar tag real.
- Páginas legales base pendientes de revisión jurídica.
